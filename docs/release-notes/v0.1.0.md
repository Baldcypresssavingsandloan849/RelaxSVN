# RelaxSVN v0.1.0

Initial public release.

## Highlights

- Modern macOS SVN desktop client for Apple Silicon
- Manage multiple working copies without repository count limits
- Checkout remote repositories
- Refresh local status and check remote updates
- Update, commit, revert, delete, and ignore paths
- Browse remote directories
- View SVN commit history
- Open projects, diffs, and conflict merge flows in Visual Studio Code
- Manage SVN credentials in the app
- Detect SVN, Homebrew, and VSCode
- Install SVN through Homebrew when available
- Support legacy TLS / certificate retry options for older SVN servers

## Download

Download `RelaxSVN-0.1.0-arm64.dmg`, open it, and drag RelaxSVN to Applications.

## Requirements

- macOS on Apple Silicon
- System SVN command line tool
- Visual Studio Code is recommended for diff and conflict merge workflows

## Signing

The DMG is Developer ID signed, notarized by Apple, and stapled.
